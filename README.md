# YouTube Video Downloader

## Project Description

This project was developed using Python and Tkinter.

The application allows users to download YouTube videos by entering a video URL.

## Features

- User Friendly GUI
- Download YouTube Videos
- High Resolution Download
- Error Handling
- Folder Selection

## Technologies Used

- Python
- Tkinter
- Pytube

## Installation

```bash
pip install -r requirements.txt
```

## Run

```bash
python main.py
```
